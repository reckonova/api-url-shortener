URL Shortener Microservice

Instructions:

  1. POST an URL to [project_url]/api/shorturl/new and receive a shortened URL in the JSON response.
    Example : {"original_url":"www.google.com","short_url":1}
.

Short URL Creation

  1. Invalid URLs (ex http(s)://www.example.com(/more/routes) ) will look like a JSON response with an error like {"error":"invalid URL"}

      HINT: to be sure that the submitted url points to a valid site you can use the function dns.lookup(host, cb) from the dns core module.


Example Usage:

  1. When I visit the shortened URL, it will redirect me to my original link.
      Example: https://1fcc-url-shortener-dcc.glitch.me/api/shorturl/898
        Will Redirect to: https://forum.freecodecamp.com
